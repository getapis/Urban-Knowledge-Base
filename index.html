GitHub Sync

Content is saved by committing data/articles.json and data/categories.json to your GitHub repository. Use a fine-grained token with contents read/write access for this repository only. The token is kept for this browser session and is not written to the project.

Without GitHub sync, online edits are blocked so content is not accidentally saved only in browser memory.
